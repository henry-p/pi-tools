# pi-tools

Personal Pi package for extensions, skills, prompts, and themes.

## Contents

- `extensions/peekaboo.ts` — dormant Peekaboo CLI integration. Run `/peekaboo` in Pi to enable the `peekaboo` tool for the current session; run `/peekaboo off` to disable it.

## Install

```bash
pi install git:git@github.com:henry-p/pi-tools.git
```

Or load directly while developing:

```bash
pi -e ./extensions/peekaboo.ts
```
